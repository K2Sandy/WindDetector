# WindDetector
We make a wind monitoring system that measures wind speed and direction. The system displays the data on an LCD and gives a warning using LEDs. When the wind speed exceeds a certain limit, the system automatically sends an SMS alert using a GSM module to inform the user.

We obtained the basic wind sensor code from the manufacturer of the wind detector. The code was then modified and integrated with additional components such as a GSM SIM800L module, LEDs, a siren, and an LCD display to create a complete early warning system.

With this project, we successfully won the Gold Medal at the International Indonesia Investor Day 2024.
